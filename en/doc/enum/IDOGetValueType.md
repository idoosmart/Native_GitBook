# Health data type

### IDOGetValueType

| Type | Description |
| ---------- | ------------ |
| pedometer | Number of steps for the day |
| calorie | Calories for the day |
| heartRate | Last measured heart rate for the day |
| spO2 | Blood oxygen for the day |
| kilometer | Distance for the day (meters) |
| swimmingDistance | Indoor swimming distance for the day (meters) |
 | sleepScore | Sleep score for the day | 
 | runningCount | Number of runs for the day | 
 | swimmingCount | Number of swims for the day | 
 | dayWorkoutCount | Number of exercises for the day | 
 | weekWorkoutCount | Number of exercises for the week | 
 | bodyBattery | Body battery |
