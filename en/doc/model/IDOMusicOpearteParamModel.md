# `IDOMusicOpearteParamModel`

| folderOperate | Int | Folder (playlist) operation<br/>0: Invalid operation<br/>1: Delete folder<br/>2: Add folder<br/>3: Modify playlist<br/>4: Import playlist<br/>5: Delete music|
