IDOTakeMedicineRemindInfoItemModel

IDOTakeMedicineRemindInfoItemModel

Single take-medicine reminder item.

Property Type Description
index Int? Item index from 1
year / month / day Int? Date
repeatValue Int? bit0: master repeat; bit1~bit7: Mon–Sun
name String? Medicine name
dailyTimes Int? Daily dose count
timeCount Int? Number of reminder times
timeItems List? Reminder time list

JSON key: repeat (Kotlin property repeatValue).
