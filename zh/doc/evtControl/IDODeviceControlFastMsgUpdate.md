### 主动通知APP短信


**App下发的json字段**:

| 字段名     | 字段类型 | 字段说明                            |
| ---------- | -------- | ----------------------------------- |
| is_success | int      | 0app发送信息失败，1app发送信息成功  |
| msg_ID     | int      | 回复的ID :每个消息对应一个ID        |
| msg_type   | int      | 消息类型                            |
| msg_notice | int      | 0是没有对应的短信回复，对应回复列表 |

`示例：`

```c
{
  "is_success":0,
  "msg_ID":1,
  "msg_type":1,
  "msg_notice":0
}
```

**App收到的json字段**：

| 字段名     | 字段类型 | 字段说明                                   |
| ---------- | -------- | ------------------------------------------ |
| msg_type   | int      | 回复的消息类型：来电短信（1）  QQ，微信 |
| msg_ID     | int      | 回复的ID :每个消息对应一个ID               |
| msg_notice | int      | 0是没有对应的短信回复，对应回复列表        |

`示例：`

```c
{
  "msg_ID":1,
  "msg_type":1,
  "msg_notice":0
}
```
