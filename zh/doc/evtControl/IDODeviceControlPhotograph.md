### 设备控制APP相机


**App收到的json字段**：

| 字段名 | 字段类型 | 字段说明                                                     |
| ------ | -------- | ------------------------------------------------------------ |
| type   | int      | 0：打开相机 <br />1：点击开始拍照 <br />2：拍照结束 <br />3：请求系统配对 |

`示例：`

```c
{
    "type":0
}
```
