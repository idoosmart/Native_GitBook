# 连接错误

### IDOConnectErrorType

| 类型                   | 说明              |
| ---------------------- | ----------------- |
| none                   | 无状态            |
| abnormalUUIDMacAddress | UUID或Mac地址异常 |
| bluetoothOff           | 蓝牙关闭          |
| connectCancel          | 主动断开连接      |
| fail                   | 连接失败          |
| timeOut                | 连接超时          |
| serviceFail            | 发现服务失败      |
| characteristicsFail    | 发现特征失败      |
| pairFail               | 配对异常          |
| informationFail        | 获取基本信息失败  |
