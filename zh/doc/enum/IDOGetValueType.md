# 健康数据类型

### IDOGetValueType

| 类型       | 说明         |
| ---------- | ------------ |
| pedometer | 当天步数 |
| calorie | 当天卡路里 |
| heartRate | 当天最近一次测量的心率 |
| spO2 | 当天血氧 |
| kilometer | 当天距离（米） |
| swimmingDistance | 当天室内游泳距离（米） |
| sleepScore | 当天睡眠得分 |
| runningCount | 当天跑步次数 |
| swimmingCount | 当天游泳次数 |
| dayWorkoutCount | 当天运动次数 |
| weekWorkoutCount | 当周运动次数 |
| bodyBattery | 身体电量 |
