# `IDOUnitModel`

| 属性        | 类型    | 说明         |
| ----------- | ------- | ------------ |
| distUnit            | Int  | 距离单位：<br/>0：无效<br/>1：公里（公制）<br/>2：英里（英制） |
| swimPoolUnit | Int  | 泳池单位设置：<br/>0：无效<br/>1：默认米<br/>2：码           |
| temp                | Int  | 温度单位：<br/>0：无效<br/>1：℃<br/>2：℉                     |
| language            | Int  | [语言](../IDOLanguageCode.md)                                |
