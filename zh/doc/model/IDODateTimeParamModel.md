# `IDODateTimeParamModel`

| 属性        | 类型    | 说明         |
| ----------- | ------- | ------------ |
| year | Int | 年 |
| monuth | Int | 月 |
| day | Int | 日 |
| hour | Int | 小时 |
| minute | Int | 分钟 |
| second | Int | 秒 |
| week | Int | 周：周一至周日 0-6 |
| timeZone | Int | 24 小时格式的时区：0-12 为东，13-24 为西 (未启用，赋值0) |
