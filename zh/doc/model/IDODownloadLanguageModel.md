# `IDODownloadLanguageModel`

| 属性        | 类型    | 说明         |
| ----------- | ------- | ------------ |
| useLang | Int | 当前使用的语言 |
| defaultLang | Int | 默认语言 |
| fixedLang | Int | 固定存储语言数量 |
| maxStorageLang | Int | 最多存储语言 |
| langArray | List<IDODownloadLanguageType> | [IDODownloadLanguageType]() 集合 |
