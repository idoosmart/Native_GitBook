## 关于SDK full 和 lite版本区别： 

full - 内置了IDO蓝牙库，lite -  无内置蓝牙库（有使用自己蓝牙库需求的用户选择），full和lite使用相同在线文档。

full 和 lite 二选一即可，当使用lite版时，注意文档中 "(仅限lite使用)" 字样提示。
